제1조 (기본 원칙)

본 서비스는 후보자별 공약 자료 등록, AI 응답 구조 세팅, 후보자 페이지 구성, 대시보드 제공 등 맞춤형으로 구축되는 서비스입니다.

따라서 결제 이후 작업 진행 단계에 따라 환불 기준이 달라집니다.

제2조 (환불 기준)

결제 이후 작업 진행 단계에 따라 환불 비율이 다음과 같이 적용됩니다.

① 시스템 세팅 시작 전
전액 환불
후보자 자료 등록·공약 세팅·AI 응답 구조 설정·후보자 페이지 구성 등 작업이 시작되기 전
② 시스템 세팅 시작 후
50% 환불
후보자 자료 분석·공약 등록·AI 응답 구조 설정·후보자 페이지 구성 등 맞춤형 작업이 시작된 경우
③ 서비스 오픈 후
환불 불가
후보자 페이지 개설·유권자 응답 기능 활성화·대시보드 제공 중 하나라도 완료된 시점부터

① 시스템 세팅 시작 전

후보자 자료 등록, 공약 세팅, AI 응답 구조 설정, 후보자 페이지 구성 등 작업이 시작되기 전에는 전액 환불됩니다.

② 시스템 세팅 시작 후

후보자 자료 분석, 공약 등록, AI 응답 구조 설정, 후보자 페이지 구성 등 맞춤형 작업이 시작된 경우에는 결제금액의 50%만 환불됩니다.

③ 서비스 오픈 후

후보자 페이지 개설, 유권자 응답 기능 활성화, 대시보드 제공 중 하나라도 완료된 경우 서비스 오픈으로 보며, 이후에는 환불이 불가합니다.

제3조 (대화카드 / 추가 응답량 환불)

대화카드 및 추가 응답량은 사용량 기반 상품입니다.

  • 구매 후 사용이 시작된 경우 환불이 불가합니다.
  • 단, 미사용 상태인 경우에 한해 결제금액의 50% 환불이 가능합니다.

제4조 (환불 불가 사유)

다음 사유는 환불 대상이 아닙니다.

  • 후보자 또는 캠프의 단순 변심
  • 자료 제출 지연
  • 후보자 일정 변경
  • 선거 전략 변경
  • 후보 등록 취소 또는 사퇴
  • 캠프 내부 사정
  • 제공된 자료의 오류로 인한 수정 요청
  • 서비스 오픈 이후 환불 요청

제5조 (회사 귀책 사유)

회사의 명백한 귀책으로 서비스 제공이 불가능한 경우, 제공 내역과 진행 상태를 기준으로 환불 또는 보상 범위를 협의합니다.

단, 다음과 같이 회사가 통제할 수 없는 사유는 회사 귀책으로 보지 않습니다.

  • 외부 API 장애
  • 클라우드 장애
  • 통신 장애
  • 선거관리기관 정책 변경
  • 법령 변경

결제 수단별 환불 처리

  • 가상계좌 — 후보자가 등록한 환불 계좌로 이체 (영업일 기준 3~5일)
  • 카드 — 결제 카드로 자동 승인 취소 (영업일 기준 3~7일)
  • 계좌이체 / 간편결제 — 원 결제 수단으로 환불 (영업일 기준 3~5일)

결제 수단별 환불 비율(전액·50%·불가) 기준은 위 제2조와 동일하게 적용됩니다.

제6조 (환불 신청 방법)

환불 요청은 공식 이메일(ilkkun.official@gmail.com)을 통해 접수해야 합니다.

환불 요청 시, 신속한 처리를 위해 아래 정보를 함께 기재해 주시기 바랍니다.

  • 성함
  • 핸드폰번호
  • 선택한 플랜

회사는 접수된 내용을 기준으로 확인 후 환불 가능 여부 및 절차를 안내합니다.

제7조 (동의 문구)

결제자 동의 사항

결제자는 본 서비스가 후보자별 맞춤 구축형 서비스이며, 시스템 세팅 시작 후에는 50% 환불, 서비스 오픈 후에는 환불이 불가함을 확인하고 동의합니다.
