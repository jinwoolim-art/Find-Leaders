## 1. 정체성

> **"AI 중재 소개팅 — 별이가 자리를 만들어주는 서비스"**

기존 소개팅 앱 (글램·아만다·정오의데이트·골드스푼)과의 차별:

1. **AI 중재** — 별이가 대화·매개·후기 운영. 사용자 대화 부담 ↓
2. **호기심·설레임 게임화** — 매칭 전 단계가 별이 키우기 (락인)
3. **양방향 이슈 매칭** — 외모 X, 성향·답변 매칭
4. **장점 자동 소개** — 사용자 자랑 부담 ↓ (한국 정서)
5. **후기 중재** — 만남 후 부드러운 피드백·다음 매칭 학습

---

## 2. 매커닉 — 10 단계 (탐님 컨셉 2026-05-28)

```
1. 등록 + 나이·성향 입력
   ↓
2. 별이가 사용자와 소통 → 성향 구체적 파악
   - 자유 텍스트 또는 시스템 카드 (별 §17 진화 게이트 패턴)
   - 사용자는 설레임 (호기심 락인)
   - AI 아바타가 사용자 닮아가며 자람 (별이 5각 차트 응용)
   ↓
3. 매칭 가능성 사람 나타남
   - 별이가 성향 일치도 분석
   ↓
4. 별이가 양쪽에 양방향 이슈 던짐
   - 두 사람 모두 답변 → 답변 일치도 측정
   ↓
5. 가장 가까운 매칭 우선순위 + 다수 풀 유지
   - 마지막까지 재시도 가능
   ↓
6. 아바타 교환
   - 인사·소개 단계 (사진 X, AI 아바타만)
   ↓
7. 1:1 대화 필요시 열림
   ↓
8. 별이가 대화 중재 + 장점 자동 소개
   - 이미 입력된 장점 자연스럽게 노출
   - 부드러운 분위기 (소개팅 자리)
   ↓
9. 만남 결정 (양방향 동의 시)
   - 만남 조율 (장소·시간·비용·코스) — Mediate 매개 매커닉
   ↓
10. 두 사람 후기 별이가 중재
    - 만남 후 피드백 양방향 정리
    - 다음 매칭 학습
```

## 4. 사업성 분석

### 시장
- 한국 소개팅 앱 시장 (글램·아만다·정오의데이트·골드스푼) — 연 매출 수백억대
- 매칭 앱 사용자 — 20-40대 (광범위)

### 차별점
- AI 중재 — 한국 정서 (대화 부담·자랑 부담) 직접 해결
- 호기심 단계 게임화 — 매칭 전부터 락인
- 외모 X 성향 매칭 — 윤리 정렬

### 수익 모델
- 가입·호기심 단계 **무료** (락인 동력)
- 매칭 unlock **Pro ₩9,900-19,900/월**
- 만남 조율 Pro+ ₩29,900/월 옵션
- 아바타 의상·시즌 이벤트·특별 카드 in-app 결제
- 매칭 성공 수수료 X (사용자 부담 ↓ → 정통 데이팅 앱 인상 회피)

### 정체성 위치
- "AI 중재 도구" 정체성 유지 — 법적 분류·정통 데이팅 앱 의무 회피
- v1.0: 멘토·동행·취미 친구 매칭 (안전한 시작)
- v1.x: 연애·소개팅 매칭 (법적 검토 후)
