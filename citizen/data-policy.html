시민 데이터 처리 기준 필수

시행일 2026년 5월 13일 · 제공자 (주)플레이4

일꾼을묻다는 시민의 주민등록번호·전화번호·정확한 GPS 위치를 수집하지 않습니다. 후보자나 캠프에는 시민 개인을 식별할 수 있는 정보가 제공되지 않으며, 비식별 통계 형태로만 제공됩니다.

회사가 수집하지 않는 것

  • 주민등록번호
  • 전화번호
  • 정확한 GPS 위치정보 (시민이 직접 선택한 행정구역만 사용)
  • 금융 정보·계좌·결제 정보 (시민 이용료가 없으므로 불필요)

회사가 수집·저장하는 것

아래 정보는 서비스 제공과 운영을 위해 수집·저장됩니다.

  • 시민이 직접 선택한 지역 정보 (시·도 / 시·군·구)
  • 후보자 조회 기록, 공약 조회 기록
  • 관심 후보 저장 내역, 공감 클릭 기록
  • 후보자에게 보낸 질문 내용 및 대화 기록
  • 앱·웹 이용 기록 (접속 시간, 페이지 흐름 등)
  • 소셜 로그인 식별값 및 이메일 (계정 식별 최소 정보)

후보자·캠프에 제공되는 형태

시민 데이터는 후보자 또는 캠프에 개인 식별 가능한 형태로 제공되지 않습니다. 다음과 같은 비식별 통계만 제공됩니다.

  • 질문 키워드 분포
  • 공약 주제별 공감 추이
  • 지역별·연령대별 관심 흐름
  • 미답변 질문 수

사용 목적 — 사용 금지 항목

회사는 시민 데이터를 다음 목적으로 사용하지 않습니다.

  • 광고 타겟팅
  • 지지율 조사·여론조사
  • 당선 예측·당락 예상
  • 제3자 마케팅 목적의 제공

주의 사항

  • 시민이 질문 내용에 본인의 이름·연락처·주소 등을 직접 입력한 경우, 해당 정보가 질문 내용에 포함될 수 있습니다. 개인정보를 질문 본문에 적지 않도록 주의해 주세요.
  • 개인정보 처리에 관한 자세한 내용은 개인정보 처리방침을 따릅니다.
